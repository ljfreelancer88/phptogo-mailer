Sending bulk email in background from PHP to Go.

## Installation

You can install phptogo using `go get`:

```bash
go get -u github.com/ljfreelancer88/phptogo-mailer@latest
```

## Usage

```go
import "github.com/ljfreelancer88/phptogo-mailer"
```

### Testing

```bash
go test -v
```
